# CPP-UDP-bmi-calculator
<b>To use: </b>
To test the program on one computer, you can move two cpp files into two projects that will be located in the same solution.
Then:
1) Left-click on your solution
2) Properties
3) And you configure the solution launch so that you launch 2 projects together

# The essence of the project
A simple example of calculating body BMI (UDP_SOCKET)
